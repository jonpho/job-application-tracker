# Job Application Tracker

This application was created to make tracking your submitted job applications easier.

## Tech Stack

* Ruby on Rails
* SQLite
* React-Bootstrap (TODO)
* Docker (TODO)

## Directions to Run

1. `bundle install`
2. `rails db:create`
3. `rails db:migrate`
4. `rails start`
5. Navigate to `http://localhost:3000/companies`
